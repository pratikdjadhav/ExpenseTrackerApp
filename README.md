# 💰 Java Expense Tracker Console App

A fully functional **console-based expense tracker** built with Java.
This app allows users to record transactions, view history, check balance summaries, and persist data using file handling.

---

## 📸 Screenshot

![App Screenshot](Screenshot (13).png)

---

## 🔧 Features

- ➕ Add income or expense transactions
- 📋 View all transactions
- 💵 Get a summary (total income, expense, balance)
- ❌ Delete a transaction by ID
- 💾 Auto save/load from `transactions.txt`
- 📆 Automatically fills in today’s date

---

## 🧱 Technologies Used

- Java (JDK 8+)
- Core OOP (classes, methods, encapsulation)
- `ArrayList`, `Scanner`, `BufferedReader/Writer`
- `LocalDate` for date handling
- File-based data storage

---

## 📁 File Format Example (`transactions.txt`)

```
1 | 2025-07-13 | Income | 1000.0 | Salary | Freelance payment
2 | 2025-07-13 | Expense | 500.0 | Food | Dinner
````

Each record is saved in this structure:

```
ID | Date | Type | Amount | Category | Note
```

## ✅ How to Run

1. Clone/download this repository
2. Open in your favorite IDE (e.g., VS Code, IntelliJ)
3. Compile & run:


javac Transaction.java Main.java
java Main


4. The app will create/update `transactions.txt` for storage
